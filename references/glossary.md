## 代码层面落地

在报告生成脚本中导入术语处理函数：

```python
# 术语首次使用追踪器
_GLOSSARY_USED = set()

GLOSSARY = {
    "Cronbach's α": {'first': '克朗巴赫α信度系数(Cronbach\'s α)', 'short': 'α系数'},
    'KMO':          {'first': 'KMO取样适当性度量',                 'short': 'KMO值'},
    'Bartlett':     {'first': '巴特利特球形检验(Bartlett)',          'short': 'Bartlett检验'},
    'Pearson':      {'first': '皮尔逊积差相关(Pearson)',            'short': 'Pearson相关'},
    'Spearman':     {'first': '斯皮尔曼等级相关(Spearman)',         'short': 'Spearman相关'},
    'ANOVA':        {'first': '方差分析(ANOVA)',                    'short': '方差分析'},
    'Bootstrap':    {'first': '自抽样法(Bootstrap)',                 'short': 'Bootstrap法'},
    'VIF':          {'first': '方差膨胀因子(VIF)',                   'short': 'VIF'},
    'OLS':          {'first': '普通最小二乘回归(OLS)',               'short': 'OLS回归'},
    # ... 按需扩展
}

def g(term_en):
    """术语智能翻译：首次出现返回中文全称(英文)，后续返回简称"""
    if term_en not in _GLOSSARY_USED:
        _GLOSSARY_USED.add(term_en)
        return GLOSSARY.get(term_en, {}).get('first', term_en)
    return GLOSSARY.get(term_en, {}).get('short', term_en)

def reset_glossary():
    """新报告开始时重置"""
    _GLOSSARY_USED.clear()
```
