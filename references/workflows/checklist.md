# 通用检查清单（所有工作流共用）

## 开工前检查（Before）

- [ ] 原始数据已备份到 `原始数据/`
- [ ] 数据编码核查（分类变量含义确认，无反转）
- [ ] 缺失值统计 → 决定处理策略
- [ ] 变量类型确认（连续/分类/有序）
- [ ] 根据决策树选定分析方法
- [ ] 正态性检验（决定参数/非参数）
- [ ] 方差齐性检验（t检验/ANOVA前）
- [ ] VIF检验（回归前）
- [ ] 交付物目录建好（`交付成果/`）

## 交付前核查（After）—— 🔴 铁律级

- [ ] **每张分析表都紧跟一段文字解读**
  - 人口学表 → 解读样本结构
  - 信度表 → 解读α值
  - 效度表 → 解读KMO/Bartlett
  - 相关表 → 解读关键变量关系
  - 差异表 → 解读显著组别
  - 回归表 → 解读系数+假设验证
  - 中介表 → 解读a/b/c/c'路径、Bootstrap置信区间
- [ ] 表格编号连续（表1、表2...无跳号无重复）
- [ ] 所有表格为三线表格式
- [ ] 图表编号连续，图题在图下方
- [ ] Word 字体正确（宋体+TNR）
- [ ] 文字解读风格去AI味
- [ ] 英文术语首次出现附中文全称（参见术语对照表）
- [ ] 源代码已整理并保留
- [ ] 绘图代码已沉淀到 `plot_templates/`
